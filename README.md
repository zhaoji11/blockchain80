## 📖 项目简介

本项目为《开源文化》课程的期末考查作业，旨在通过实践掌握GitHub项目管理、版本控制协作以及开源战略分析等核心技能。

**项目目标**：
- 熟练使用GitHub进行项目管理和团队协作
- 理解开源许可证的选择与应用
- 掌握分支管理和冲突解决
- 应用华为"五看三定"模型进行项目战略分析
